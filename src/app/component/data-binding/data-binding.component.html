Data binding is a mechanism to coordinate the communication between a component and its view.

There are four types of data binding in Angular:

  • Interpolation
  • Property Binding
  • Event Binding
  • Two-way Binding

Interpolation

Interpolation is a one-way data binding technique that allows you to bind data from the component class to the template.

It is done using double curly braces.
